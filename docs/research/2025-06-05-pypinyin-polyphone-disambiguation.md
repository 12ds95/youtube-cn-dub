# pypinyin 多音字消歧方案迁移

**日期**: 2025-06-05
**状态**: 已实施

## 背景

edge-tts 不支持 SSML `<phoneme>` 标签（`Communicate` 内部转义 XML，标签会被朗读），
只能通过文本级同音字替换纠正多音字误读。

## 旧方案：正则规则（已废弃）

手工编写 15 条 regex 规则，覆盖 13 个高频多音字。
每条规则匹配特定词组上下文，替换为同音字。

**局限**:
- 覆盖面受限于手工列举的词组（如 `重+新/复/建/...`）
- 新增多音字需手写正则，容易遗漏上下文
- 无法处理规则未覆盖的罕见搭配

**已废弃的 `_POLYPHONE_RULES` 代码**:

```python
_POLYPHONE_RULES: List[tuple] = [
    # 了 (le/liao): 了解/了然/了不起 → 瞭
    (re.compile(r"了(解|然|不起|无|如指掌|结|事|断|却|得|望)"), r"瞭\1"),
    # 得 (de/de/dei): no-op, edge-tts handles correctly
    (re.compile(r"(获|取|觉|值|舍|记)(得)"), r"\1\2"),
    (re.compile(r"得(亏|靠)"), r"得\1"),
    # 行 (hang/xing): 银行/行业 → 杭
    (re.compile(r"(银|央|商|投|同|内|外|在)(行)(?![动进走为驶驰了])"), r"\1杭"),
    (re.compile(r"行(业|列|情|号|距|家|当|规)"), r"杭\1"),
    # 数 (shu/shu): 数一数二 → 属
    (re.compile(r"数(一数二|不胜数|落|说)"), r"属\1"),
    # 重 (zhong/chong): 重新/重复 → 虫
    (re.compile(r"重(新|复|建|启|来|试|置|写|装|做|现|返|叠|演|申|审|组|设|排|整|定义|命名|构|塑|回|制|载|开|发|拨|提|归)"), r"虫\1"),
    # 处 (chu/chu): 到处/各处 → 触
    (re.compile(r"(到|各|用|好|坏|长|短|何|深|远|近|随|四|别|妙|益)(处)"), r"\1触"),
    # 调 (diao/tiao): 调整/协调 → 条
    (re.compile(r"(协)(调)"), r"\1条"),
    (re.compile(r"调(整|节|解|和|配|谐|制|控|理|频|幅)"), r"条\1"),
    # 率 (lv/shuai): 率领/率先 → 帅
    (re.compile(r"率(领|先|队|部|军|众|性|直|真)"), r"帅\1"),
    # 量 (liang/liang): 测量/丈量 → 良
    (re.compile(r"(测|丈|衡|估|计|度|称)(量)"), r"\1良"),
    # 传 (chuan/zhuan): 自传/外传 → 撰
    (re.compile(r"(自|外|列|内|经|别|正|小|评)(传)(?![输送播递达承感染导])"), r"\1撰"),
    # 应 (ying/ying): 应该/应当 → 英
    (re.compile(r"应(该|当|有尽有|许|属|予)"), r"英\1"),
    # 乐 (le/yue): 音乐/乐器 → 月
    (re.compile(r"(音)(乐)(?!趣|观)"), r"\1月"),
    (re.compile(r"乐(器|曲|队|谱|章|团|坛|理|律|感|手)"), r"月\1"),
    # 的 (de/di): 的确 → 滴
    (re.compile(r"的(确|当)"), r"滴\1"),
    # 差 (cha/chai): 出差/差遣 → 拆
    (re.compile(r"(出)(差)(?!异|距|别|值|额|价|分|评|错)"), r"\1拆"),
    (re.compile(r"差(遣|事|役|使)"), r"拆\1"),
]
```

## 新方案：pypinyin 短语级消歧

用 [pypinyin](https://github.com/mozillazg/python-pinyin) 词组词典自动判定上下文读音，
配合 `_POLYPHONE_HOMOPHONE_MAP` 字典做定向替换。

**优势**:
- 自动覆盖所有词组上下文（不限于手工列举）
- 新增多音字只需在 map 中加一行
- pypinyin 活跃维护，词组词典持续更新

**数据结构**:

```python
_POLYPHONE_HOMOPHONE_MAP: Dict[str, Dict[str, str]] = {
    '了': {'liǎo': '瞭'},   '行': {'háng': '杭'},
    '重': {'chóng': '虫'},   '调': {'tiáo': '条'},
    '率': {'shuài': '帅'},   '量': {'liáng': '良'},
    '传': {'zhuàn': '撰'},   '应': {'yīng': '英'},
    '乐': {'yuè': '月'},     '的': {'dí': '滴'},
    '差': {'chāi': '拆'},    '数': {'shǔ': '属'},
    '处': {'chù': '触'},
}
```

**对齐算法**:
pypinyin 将连续非汉字（英文/数字/URL）合并为单个条目，
用条目文本长度跳过非汉字序列，汉字 1:1 对齐。

**性能**: ~0.5ms/句（45 字），含快速路径跳过（无多音字时 0 开销）。

## 废弃 fallback 的理由

pypinyin 纯 Python 无 C 扩展，与 jieba 同级别轻量依赖。
项目通过 venv 管理依赖，不存在安装困难。无需保留 regex fallback。
